# Developer Documentation

This section contains documentation for developers contributing to or extending the Qwen3-VL multimodal model. Here you'll find architecture details, development practices, and technical implementation information.

## Contents

- [Architecture](architecture/README.md) - System design and architectural decisions
- [Development Guides](guides/README.md) - Information for setting up development environment and practices
- [Testing](testing/README.md) - Testing strategies and procedures

## Audience

This documentation is designed for:
- Core developers working on the Qwen3-VL codebase
- Contributors looking to understand the system architecture
- Developers extending the model's functionality

## Getting Started

If you're new to developing for Qwen3-VL, start with the [Architecture Overview](architecture/README.md) to understand the system design, then check the [Development Setup Guide](guides/development/README.md) for environment configuration.
