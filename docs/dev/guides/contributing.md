# Contribution Guidelines

For the most up-to-date contribution guidelines, please refer to our [CONTRIBUTING.md](../../../CONTRIBUTING.md) file in the project root.

## Overview

Our contribution process includes:

- Following the code of conduct
- Setting up your development environment
- Understanding the project structure
- Adhering to coding standards
- Writing and running tests
- Submitting changes via pull requests

## Getting Started

To contribute to the Qwen3-VL project:

1. Read our [Code of Conduct](../../../CODE_OF_CONDUCT.md)
2. Set up your [Development Environment](development/README.md)
3. Familiarize yourself with the [Project Structure](../../README.md#structure-overview)
4. Follow our [Coding Standards](../README.md#coding-standards)
5. Review our [Testing Guidelines](../testing/README.md)

## Submitting Changes

All contributions are submitted through GitHub pull requests. Please follow the guidelines in our main [CONTRIBUTING.md](../../../CONTRIBUTING.md) file for detailed instructions on the submission process.
